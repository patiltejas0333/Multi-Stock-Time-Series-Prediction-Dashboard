A Streamlit-based interactive dashboard for analyzing and forecasting stock prices using AR, MA, ARMA, and ARIMA models with ACF & PACF visualization.

## 🚀 Features
📊 Multi-stock comparison (NSE stocks)
📉 ACF (Autocorrelation) visualization
📉 PACF (Partial Autocorrelation) visualization
🔮 Time series forecasting:
AR (Auto Regression)
MA (Moving Average)
ARMA
ARIMA
📅 30-day future stock prediction
📈 Interactive UI using Streamlit
🌐 Live stock data using Yahoo Finance (yfinance)
🛠️ Tech Stack
Python
Streamlit
Pandas & NumPy
Matplotlib
Statsmodels
yFinance

## 📊 Usage
Select one or multiple stocks (e.g., RELIANCE.NS, TCS.NS)
Choose date range
Select model (AR / MA / ARMA / ARIMA)
Analyze:
Stock trends
ACF & PACF plots
View 30-day forecast predictions
